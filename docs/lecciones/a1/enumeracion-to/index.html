Iteración 1: と para listas completas

Misión: Aprender a enumerar todos los elementos de una lista usando la partícula と.

📐 La partícula と enumerativa

La partícula と se usa para enumerar todos los elementos de una lista de forma exhaustiva. Equivale a «y» en español, pero con la diferencia de que implica que no faltan elementos en la lista.

  • Sustantivo1 + と + Sustantivo2 + と + Sustantivo3
  • La lista puede tener 2 o más elementos
  • Se puede omitir el último と antes del verbo, pero es más claro incluirlo

📐 Estructura

[Elemento1] と [Elemento2] と [Elemento3] (と) [partícula de caso] [verbo]

💬 Ejemplos

JaponésTraducción
りんごとバナナとみかんを買かいましたCompré manzana, plátano y mandarina
犬いぬと猫ねこがいますHay un perro y un gato
日に本ほんと中ちゅう国ごくと韓かん国こくに行いきましたFui a Japón, China y Corea
先せん生せいと友とも達だちと食しょく事じをしましたComí con el profesor y mi amigo

💡 Diferencia clave

  • と → lista completa (todos los elementos están nombrados)
  • や → lista incompleta (solo se mencionan ejemplos, hay más)

📝 Práctica

Traduce al japonés:

1. Compré un libro y un bolígrafo

2. Mi padre y mi madre están en casa

3. El sushi y el sashimi son deliciosos

Iteración 2: Resumen y glosario

Misión: Repasar los usos de と y consolidar el vocabulario.

📐 Resumen

Usoと (exhaustivo)
SignificadoLista completa de elementos — todos están nombrados
EstructuraA と B と C
Partícula al finalOpcional, pero recomendable
Contrasteや = lista incompleta (ejemplos)

📖 Glosario de la lección

Ver términos nuevos - **と** — partícula enumerativa exhaustiva («y») - **りんご** — manzana - **バナナ** — plátano - **みかん** — mandarina - **買(か)いました** — compré (forma pasada de 買います) - **犬(いぬ)** — perro - **猫(ねこ)** — gato - **〜がいます** — hay (seres animados) - **日(に)本(ほん)** — Japón - **中(ちゅう)国(ごく)** — China - **韓(かん)国(こく)** — Corea - **行(い)きました** — fui (forma pasada de 行きます) - **先(せん)生(せい)** — profesor(a) - **友(とも)達(だち)** — amigo/a - **食(しょく)事(じ)をしました** — comí (forma pasada de 食事をします) - **父(ちち)** — padre - **母(はは)** — madre - **家(いえ)** — casa - **寿(す)司(し)** — sushi - **刺(さ)身(しみ)** — sashimi - **美味(おい)しい** — delicioso/a - **や** — partícula enumerativa no exhaustiva («y, entre otros»)
