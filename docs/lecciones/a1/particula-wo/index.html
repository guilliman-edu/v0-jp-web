Iteración 1: ¿Qué es el complemento directo?

Misión: Entender qué es un complemento directo (CD) antes de ver cómo se marca en japonés.

🎯 Concepto: el complemento directo

El complemento directo es la parte de la oración que recibe directamente la acción del verbo. No realiza la acción, la recibe.

Prueba del CD: pregúntale al verbo ¿qué? o ¿a quién?
  • Yo como una manzana → ¿Qué como? → Una manzana
  • Él lee un libro → ¿Qué lee? → Un libro
  • Ella ve una película → ¿Qué ve? → Una película

⚠️ Atención

El CD no es el sujeto. El sujeto hace la acción; el CD la recibe.

  • 私わたしは 先生せんせい です → 先生せんせい no es CD (es atributo con です)
  • 私わたしは 水みず を 飲のみます → 水みず sí es CD (recibe la acción de beber)

💬 Identificar el CD

Para reconocerlo:

1. Pregunta ¿qué? al verbo

2. Sustitúyelo por lo, la, los, las

Oración¿Qué?CD
Yo bebo agua¿Qué bebo?agua
Él escribe una carta¿Qué escribe?una carta
Estudio japonés¿Qué estudio?japonés

Iteración 2: La partícula を marca el CD

Misión: Aprender que を (wo) señala el complemento directo en japonés.

📐 La partícula を (wo)

En japonés el CD se marca con la partícula を (wo). Se escribe を pero se pronuncia o (suena como la o española).

を va detrás del CD y antes del verbo.

📐 Estructura SOV

El japonés tiene orden Sujeto + Objeto + Verbo (SOV). El CD con を va siempre antes del verbo.

Español (SVO)Japonés (SOV)
Yo bebo agua私わたしは 水みずを飲のみます
Tú lees un libroあなたは 本ほんを読よみます
Él ve una película彼かれは 映画えいがを見みます

JaponésTraducción

私わたしは 本ほん を 読よみますYo leo un libro

⚠️ Atención: を solo con verbos transitivos

を solo se usa con verbos transitivos (verbos que necesitan un CD para completar su significado). Verbos como 行いく (ir), 帰かえる (volver), 寝ねる (dormir) no llevan を.

  • ✅ 水みずを飲のみます (beber → transitivo)
  • ❌ 水みずを 行いきます (ir → intransitivo, no lleva CD)

🗣️ Conversación

A: 毎日まいにち 何なに を しますか

B: 日本語にほんご を 勉強べんきょうします

A: 週末しゅうまつ 映画えいが を 見みますか

B: はい、見みます

Iteración 3: Negación y pasado con CD

Misión: Usar を en oraciones negativas y en pasado, combinando con la conjugación -ます.

📐 Negación: を + ません

El CD con を funciona igual en afirmativo y en negativo. Solo cambia la conjugación del verbo.

AfirmativoNegativo
お茶おちゃ を 飲のみますお茶おちゃ を 飲のみません
新聞しんぶん を 読よみます新聞しんぶん を 読よみません
宿題しゅくだい を します宿題しゅくだい を しません

📐 Pasado: を + ました / ませんでした

Afirmativo pasadoNegativo pasado
お茶おちゃ を 飲のみましたお茶おちゃ を 飲のみませんでした
映画えいが を 見みました映画えいが を 見みませんでした
宿題しゅくだい を しました宿題しゅくだい を しませんでした

💬 Ejemplos completos

JaponésTraducción
昨日きのう、映画えいが を 見みましたAyer vi una película
毎日まいにち、新聞しんぶん を 読よみますCada día leo el periódico
今晩こんばん、お茶おちゃ を 飲のみませんEsta noche no tomo té
週末しゅうまつ、友達ともだち と テレビ を 見みましたEl fin de semana vi la tele con un amigo

🔗 Conexión

Esto funciona igual que lo visto en la Iteración 2. La diferencia es solo el tiempo del verbo. El CD con を no cambia.

Iteración 4: Resumen y glosario

Misión: Repasar lo aprendido y consolidar los conceptos nuevos.

📌 Resumen

  • を marca el complemento directo (la cosa que recibe la acción)
  • El CD va antes del verbo (estructura SOV)
  • を solo se usa con verbos transitivos
  • La partícula no cambia con negación ni pasado; solo cambia el verbo
  • Verbos transitivos útiles en A1: 食たべる, 飲のむ, 読よむ, 見みる, 聞きく, 勉強べんきょうする, 書かく, 買かう, 撮とる

📖 Glosario de la lección

Ver términos nuevos
  • を(wo) — partícula que marca el complemento directo
  • 補語ほご — complemento directo: recibe la acción del verbo
  • 他動詞たどうし — verbo transitivo: necesita un CD
  • SOV — Sujeto + Objeto + Verbo: orden de palabras del japonés
