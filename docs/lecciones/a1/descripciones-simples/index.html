Iteración 1: Estructura descriptiva XはYがZ

Misión: Aprender a describir partes o características de algo con la estructura XはYが + adjetivo.

📐 Estructura XはYがZ

Para describir una parte o aspecto específico de un sujeto se usa:

`

[sujeto]は [parte/característica]が [adjetivo]

`

  • Xは — tema general de la oración
  • Yが — la parte o aspecto que se describe
  • Z — el adjetivo que describe Y

Esta estructura equivale a «X tiene Y que es Z» o «De X, Y es Z».

💬 Ejemplos

JaponésTraducción
ゾウは鼻はなが長ながいですEl elefante tiene la trompa larga
Vigoの海うみはきれいですEl mar de Vigo es bonito
彼かれは背せが高たかいですÉl es alto (lit. De él, la estatura es alta)
日本にほんは食べ物たべものがおいしいですJapón tiene comida deliciosa
このスマホは値段ねだんが安やすいですEste smartphone tiene precio barato
私わたしは頭あたまがいいですYo soy listo (lit. De mí, la cabeza es buena)

📐 Comparación con estructura simple

XはYがZ (descripción específica)XはZ (descripción general)
ゾウは鼻はなが長ながいですゾウは大おおきいです
El elefante tiene la trompa largaEl elefante es grande
彼かれは背せが高たかいです彼かれは元気げんきです
Él es altoÉl está bien

⚠️ Error común: confundir は y が

  • ❌ ゾウが鼻が長いです — suena a solo el elefante (y no otro)
  • ✅ ゾウは鼻が長いです — correcto (hablando del elefante en general)

La partícula は marca el tema; la partícula が marca el sujeto de la característica.

Iteración 2: Resumen y glosario

Misión: Repasar la estructura XはYがZ y sus usos descriptivos.

📐 Esquema general

`

Xは Yが [adjetivo]です

│ │

tema parte/característica

`

Se usa para describir:

  • Partes del cuerpo (鼻が長い, 背が高い)
  • Aspectos abstractos (値段が安い, 頭がいい)
  • Características de un lugar (海がきれい, 食べ物がおいしい)

💬 Más ejemplos

JaponésTraducción
京都きょうとはお寺てらが有名ゆうめいですKioto tiene templos famosos
彼女かのじょは目めが大おおきいですElla tiene ojos grandes
この車くるまはエンジンが新あたらしいですEste coche tiene el motor nuevo
北海道ほっかいどうは冬ふゆが寒さむいですHokkaidō tiene inviernos fríos
あの店みせはケーキがおいしいですEsa tienda tiene pasteles deliciosos
