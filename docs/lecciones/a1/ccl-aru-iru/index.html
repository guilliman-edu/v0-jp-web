Iteración 1: Verbos de existencia ある y いる

Misión: Distinguir cuándo usar ある y cuándo いる para decir "hay" o "estar".

📐 La diferencia clave

En japonés se distingue entre seres animados (personas, animales) y cosas inanimadas (objetos, plantas):

VerboTipoPara
いるGrupo IISeres animados (personas, animales)
あるGrupo ICosas inanimadas (objetos, plantas)

📐 Conjugación en forma -ます

Tiempoいる (animado)ある (inanimado)
Presente afirmativoいますあります
Presente negativoいませんありません
Pasado afirmativoいましたありました
Pasado negativoいませんでしたありませんでした

💬 Ejemplos

JaponésTraducción
猫ねこがいますHay un gato (animado)
本ほんがありますHay un libro (inanimado)
田中たなかさんは家いえにいますEl Sr. Tanaka está en casa
車くるまはあそこにありますEl coche está allí

⚠️ Error común

Usar ある para personas o いる para cosas es un error grave.

  • ✅ 人がいます (hay una persona)
  • ❌ 人があります

Iteración 2: Partícula に (lugar de existencia)

Misión: Usar に para marcar dónde existe algo.

📐 Estructura de existencia

`

[Lugar] に [sujeto] が います / あります

`

  • に marca el lugar donde algo existe
  • が marca el sujeto que existe

💬 Ejemplos

JaponésTraducción
部屋へやに机つくえがありますEn la habitación hay una mesa
公園こうえんに猫ねこがいますEn el parque hay un gato
図書館としょかんに本ほんがありますEn la biblioteca hay libros
学校がっこうに先生せんせいがいますEn la escuela hay profesores

⚠️ に vs で

  • に — lugar de existencia (algo está ahí): 庭にわに猫ねこがいます
  • で — lugar donde ocurre una acción: 庭にわで遊あそびます

Iteración 3: Partículas で (acción) y へ (dirección)

Misión: Diferenciar los usos de に, で y へ con lugares.

📐 で para lugar de acción

Cuando el lugar no es de existencia sino donde transcurre una acción, se usa で:

JaponésTraducción
図書館としょかんで勉強べんきょうしますEstudio en la biblioteca
家いえで食たべますComo en casa
公園こうえんで遊あそびますJuego en el parque

📐 へ / に para dirección

Cuando se expresa dirección (ir a, venir a), se usa へ o に:

PartículaUsoEjemplo
へdirección (se pronuncia e)学校がっこうへ行いきます
にpunto de llegada学校がっこうに行いきます

Hoy en día へ y に son intercambiables para dirección.

💬 Comparación de partículas de lugar

PartículaUsoEjemplo
にexistencia公園こうえんに猫ねこがいます
でacción公園こうえんで遊あそびます
へdirección公園こうえんへ行いきます

📖 Glosario de la lección

Ver términos nuevos
  • いる — haber/estar (seres animados)
  • ある — haber/estar (cosas inanimadas)
  • に — partícula de lugar de existencia
  • で — partícula de lugar de acción
  • へ — partícula de dirección (se pronuncia *e*)
