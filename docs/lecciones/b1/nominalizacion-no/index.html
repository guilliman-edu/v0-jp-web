Iteración 1: Subordinación con función de sustantivo

Misión: Convertir una frase completa en un sustantivo usando の.

📐 Sustantivación implícita

Cuando la subordinada modifica a un sustantivo existente, se aplica el mismo proceso que en la lección anterior:

私わたしが飲のんだミルクを彼かれも飲のんだ
"Él también bebió la leche que yo bebí"

La subordinada 私わたしが飲のんだ modifica a ミルク, y toda la frase subordinada + nombre funciona como OD (por eso lleva を).

📐 Sustantivación explícita con の

Cuando no hay un sustantivo al que modificar, se usa の como nominalizador: convierte toda la frase en un sustantivo.

"Lo que tú hiciste"
あなたがしたの

📐 Proceso

1. Identificar la subordinada

2. Pasar el verbo a forma simple

3. Cambiar は → が

4. Añadir の al final

5. Agregar la partícula según la función en la frase principal

Iteración 2: Ejemplos prácticos

Misión: Usar の para nominalizar frases completas.

💬 Proceso paso a paso

"No perdonaré lo que hiciste"

Paso 1: Dos frases:

  • あなたは何なにをしました (Tú hiciste algo)
  • 私わたしはそれを許ゆるしません (No lo perdonaré)

Paso 2: Transformar la subordinada:

  • あなたがした (hiciste → forma simple, は→が)
  • あなたがしたの (nominalización)

Paso 3: Insertar en la principal:

  • 私わたしは(あなたがしたの)を許ゆるしません

💬 Más ejemplos

JaponésTraducción
私わたしはあなたが言いったのを聞ききましたEscuché lo que dijiste
彼かれが来くるのを待まっていますEstoy esperando a que venga
私わたしが好すきなのは寿司すしですLo que me gusta es el sushi

⚠️ の nominalizador no es posesivo

No confundir este の (nominalizador) con の posesivo:

  • 私わたしの本ほん — mi libro (posesión)
  • 私わたしが読よむの — lo que leo (nominalización)

📖 Glosario

Ver términos nuevos - **名詞化(めいしか)** — nominalización - **の** — nominalizador (convierte frases en sustantivos)
