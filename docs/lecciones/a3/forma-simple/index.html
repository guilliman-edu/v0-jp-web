Iteración 1: ¿Qué es la forma simple?

Misión: Entender la forma simple (forma plana), base del lenguaje informal.

📐 Forma simple vs forma -ます

Forma -ます (cortés)Forma simple (plana)Uso
食たべます食たべるinformal, con amigos
食たべません食たべないnegativo informal
食たべました食たべたpasado informal
食たべませんでした食たべなかったpasado neg. informal

📐 Terminaciones de la forma simple

TiempoAfirmativoNegativo
Presente(forma diccionario)-ない
Pasado-た-なかった

📐 La forma indeterminada

Para construir el negativo (-ない) y el pasado (-た), se necesita la forma indeterminada, que se obtiene transponiendo el sonido -i a -a (al revés que en la forma -ます).
