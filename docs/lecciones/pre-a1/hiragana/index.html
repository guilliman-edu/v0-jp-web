Iteración 1: La tabla de hiragana

Misión: Conocer los 46 caracteres básicos del hiragana.

📐 ¿Qué es el hiragana?

El hiragana (平仮名) es uno de los tres sistemas de escritura del japonés. Es un silabario: cada carácter representa una sílaba completa.

あ aい iう uえ eお o
か kaき kiく kuけ keこ ko
さ saし shiす suせ seそ so
た taち chiつ tsuて teと to
な naに niぬ nuね neの no
は haひ hiふ fuへ heほ ho
ま maみ miむ muめ meも mo
や yaゆ yuよ yo
ら raり riる ruれ reろ ro
わ waを wo
ん n

💬 Consejos de estudio

  • Estudia de memoria una fila cada día
  • Practica la escritura a mano (orden de trazos)
  • Usa la tabla como referencia constante
